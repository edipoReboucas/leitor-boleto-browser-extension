Permissão necessária para arquivos locais

Para ler boletos em PDF ou arquivos abertos no navegador, habilite o acesso a URLs de arquivos.

  1. Abra a página de extensões do Chrome
  2. Localize LeitorBoleto
  3. Ative Permitir acesso a URLs de arquivos
  4. Volte ao PDF e clique no ícone da extensão novamente
